# Home_Plot-app-java

This is a Java-based project for managing home plot-related tasks.

## Features:
- Plot registration
- Owner management
- Payment tracking
- Admin dashboard

## Tech Stack:
- Java
- MySQL
- JSP / Servlets

## How to Run:
1. Download or clone the repository
2. Extract `homplot.zip`
3. Import into Eclipse or any Java IDE
4. Setup database using provided script
